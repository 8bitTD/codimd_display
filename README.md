## 概要

このリポジトリは、CodiMDの全文表示サイトを作成するためのソースコードです

## 使い方
[Rust](https://rust-lang.org/ja/tools/install/)をインストール後に
```Shell
git clone https://github.com/8bitTD/codimd_display
cd codimd_display
cargo run --release
```
